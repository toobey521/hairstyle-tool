📖 使用说明

  1. 方式 A（云端）：下载 HTML → 双击打开 → 配置 API Key → 上传照片 → 生成
  2. 方式 B（本地 ComfyUI）：需安装 ComfyUI + SDXL 模型 → 双击 发型设计助手.vbs 启动
  3. 选择服务商后点击「保存设置」+「测试连接」确认可用
  4. 上传正面照片 → 设置性别年龄 → 点「分析脸型 + 获取发型建议」
  5. 每款发型选数量 → 点「批量生成」→ 等待出图
  6. 点击图片放大查看，右键另存为下载
